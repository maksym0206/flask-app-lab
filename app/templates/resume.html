Про мене

Привіт! Я – програміст, який перетворює ідеї на реальність. Маю 5+ років досвіду в розробці програмного забезпечення з акцентом на створенні зручних, продуктивних і масштабованих рішень. Моє кредо – "Постійний розвиток".

Мені цікаво створювати інструменти, які допомагають бізнесу досягати нових вершин. Я особливо захоплююся оптимізацією процесів, впровадженням автоматизації та вирішенням складних задач. Окрім програмування, я активно стежу за новими трендами в IT і ділюся своїми знаннями через блоги та майстер-класи.

Освіта

  • Карпатський університет — Бакалавр комп'ютерних наук (2018-2022). Бакалаврська: розробка алгоритмів для обробки великих даних.
  • Університет Гарвард — Магістр програмної інженерії (2022-2024). Магістерська робота: створення платформи для розподілених обчислень.

Навички

  • Мови програмування: Python, JavaScript, C++
  • Фреймворки: Flask, Django, React, FastAPI
  • Розробка API: REST, GraphQL
  • Бази даних: MySQL, PostgreSQL, MongoDB
  • Інструменти DevOps: Docker, Kubernetes, Jenkins
  • Тестування: PyTest, Selenium, Postman

Досвід роботи/Проєкти

Проєкт: Інтелектуальна CRM-система

Спроектував і реалізував CRM-систему для автоматизації продажів та прогнозування клієнтських потреб. Використав Flask для серверної частини та React для інтерактивного інтерфейсу. Алгоритми машинного навчання допомогли збільшити ефективність продажів на 25%.

Проєкт: Хмарна платформа для аналітики

Розробив платформу для збору, обробки та візуалізації даних у реальному часі. Технології: Django, PostgreSQL, Kubernetes. Платформа дозволила компаніям скоротити час на прийняття рішень на 40%.

Особистий проєкт: Освітня платформа для студентів

Створив платформу, яка надає інструменти для організації навчання, тестування та відстеження прогресу. Основні функції: адаптивне тестування, інтеграція з Google Classroom, гейміфікація. Використані технології: FastAPI, React, MongoDB.
